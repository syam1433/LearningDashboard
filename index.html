Day 1 ‚Äì Introduction to C

1. What is C Language?

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Labs. It is called the ‚Äúmother of programming languages‚Äù because many modern languages (C++, Java, Python, etc.) are influenced by C.

It is fast, efficient, and powerful, mostly used in system programming, operating systems, and embedded systems.

2. Features of C

  • Simple & powerful ‚Äì easy to learn but can handle complex tasks.
  • Portable ‚Äì programs can run on different machines.
  • Structured ‚Äì programs are divided into functions.
  • Rich library ‚Äì has built-in functions.
  • Memory management ‚Äì can directly access memory using pointers.

3. Structure of a C Program

#include <stdio.h>   // Header file
int main() {         // Main function (entry point)
    printf("Hello, World!");  // Statement
    return 0;        // Exit status
}

#include <stdio.h> ‚Üí Preprocessor directive, includes standard input/output library.

main() ‚Üí Starting point of every C program.

{ } ‚Üí Denotes beginning and end of block.

printf() ‚Üí Used to print output.

return 0; ‚Üí Ends the program successfully.

4. Basic Building Blocks

  • Keywords ‚Üí Reserved words (int, return, if, for).
  • Identifiers ‚Üí Names given to variables, functions (marks, sum).
  • Constants ‚Üí Fixed values (10, 'A', 3.14).
  • Variables ‚Üí Containers to store data (int age = 20;).
  • Data Types ‚Üí Define the type of data (int, float, char, double).

5. Input & Output

#include <stdio.h>
int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);  // %d for integer
    printf("You are %d years old", age);
    return 0;
}

Day 2 ‚Äì Variables, Constants, and Data Types

üîπ 1. Variables

A variable is like a box in memory where we can store a value. Every variable has:

  • Name (identifier)
  • Type (int, float, char, ‚Ä¶)
  • Value (the data stored in it)
#include <stdio.h>
int main() {
    int age = 20;        // integer variable
    float marks = 85.5;  // float variable
    char grade = 'A';    // character variable

    printf("Age: %d\n", age);
    printf("Marks: %.2f\n", marks);
    printf("Grade: %c\n", grade);
    return 0;
}

üîπ 2. Rules for Naming Variables

  • Must start with letter or underscore.
  • Cannot use keywords like int, float, etc.
  • Cannot contain spaces or special symbols (@, #, %, ...).
  • Case sensitive (value ‚â† Value).

‚úÖ Valid Examples:

int total_marks;
float average;
char studentName[20];

‚ùå Invalid Examples:

int 2count;     // cannot start with number
float total%age; // cannot use %
char float;     // cannot use keywords

üîπ 3. Constants

Constants are values that do not change during program execution.

Types of Constants in C:

  • Integer Constants ‚Üí 10, -25, 0
  • Floating Constants ‚Üí 3.14, -0.001
  • Character Constants ‚Üí 'A', '9', '$'
  • String Constants ‚Üí "Hello", "C Language"

‚úÖ Example 1: Using const keyword

#include <stdio.h>
int main() {
    const float PI = 3.14;  // constant float
    int r = 5;
    printf("Area = %.2f\n", PI * r * r);
    return 0;
}

‚úÖ Example 2: Using #define

#include <stdio.h>
#define MAX 100   // constant defined at preprocessor level

int main() {
    printf("Maximum value = %d", MAX);
    return 0;
}

üîπ 4. Data Types in C

Data Type Size (typical) Example Range
int 4 bytes int a = 10; -2,147,483,648 to 2,147,483,647
float 4 bytes float pi = 3.14; ~6 decimal digits
double 8 bytes double big = 123.456789; ~15 decimal digits
char 1 byte char grade = 'A'; -128 to 127 or 0 to 255 
#include <stdio.h>
int main() {
    int roll = 25;
    float percentage = 89.75;
    double balance = 123456.789;
    char grade = 'A';

    printf("Roll: %d\n", roll);
    printf("Percentage: %.2f\n", percentage);
    printf("Balance: %.3lf\n", balance);
    printf("Grade: %c\n", grade);

    return 0;
}

üîπ 5. sizeof Operator

Used to find how many bytes a data type or variable occupies in memory.

#include <stdio.h>
int main() {
    printf("Size of int: %lu bytes\n", sizeof(int));
    printf("Size of float: %lu bytes\n", sizeof(float));
    printf("Size of double: %lu bytes\n", sizeof(double));
    printf("Size of char: %lu byte\n", sizeof(char));
    return 0;
}
