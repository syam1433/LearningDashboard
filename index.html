Day 1 ‚Äì Introduction to C

1. What is C Language?

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Labs. It is called the ‚Äúmother of programming languages‚Äù because many modern languages (C++, Java, Python, etc.) are influenced by C.

It is fast, efficient, and powerful, mostly used in system programming, operating systems, and embedded systems.

2. Features of C

  • Simple & powerful ‚Äì easy to learn but can handle complex tasks.
  • Portable ‚Äì programs can run on different machines.
  • Structured ‚Äì programs are divided into functions.
  • Rich library ‚Äì has built-in functions.
  • Memory management ‚Äì can directly access memory using pointers.

3. Structure of a C Program

#include <stdio.h>   // Header file
int main() {         // Main function (entry point)
    printf("Hello, World!");  // Statement
    return 0;        // Exit status
}

#include <stdio.h> ‚Üí Preprocessor directive, includes standard input/output library.

main() ‚Üí Starting point of every C program.

{ } ‚Üí Denotes beginning and end of block.

printf() ‚Üí Used to print output.

return 0; ‚Üí Ends the program successfully.

4. Basic Building Blocks

  • Keywords ‚Üí Reserved words (int, return, if, for).
  • Identifiers ‚Üí Names given to variables, functions (marks, sum).
  • Constants ‚Üí Fixed values (10, 'A', 3.14).
  • Variables ‚Üí Containers to store data (int age = 20;).
  • Data Types ‚Üí Define the type of data (int, float, char, double).

5. Input & Output

#include <stdio.h>
int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);  // %d for integer
    printf("You are %d years old", age);
    return 0;
}

Day 2 ‚Äì Variables, Constants, and Data Types

üîπ 1. Variables

A variable is like a box in memory where we can store a value. Every variable has:

  • Name (identifier)
  • Type (int, float, char, ‚Ä¶)
  • Value (the data stored in it)
#include <stdio.h>
int main() {
    int age = 20;        // integer variable
    float marks = 85.5;  // float variable
    char grade = 'A';    // character variable

    printf("Age: %d\n", age);
    printf("Marks: %.2f\n", marks);
    printf("Grade: %c\n", grade);
    return 0;
}

üîπ 2. Rules for Naming Variables

  • Must start with letter or underscore.
  • Cannot use keywords like int, float, etc.
  • Cannot contain spaces or special symbols (@, #, %, ...).
  • Case sensitive (value ‚â† Value).

‚úÖ Valid Examples:

int total_marks;
float average;
char studentName[20];

‚ùå Invalid Examples:

int 2count;     // cannot start with number
float total%age; // cannot use %
char float;     // cannot use keywords

üîπ 3. Constants

Constants are values that do not change during program execution.

Types of Constants in C:

  • Integer Constants ‚Üí 10, -25, 0
  • Floating Constants ‚Üí 3.14, -0.001
  • Character Constants ‚Üí 'A', '9', '$'
  • String Constants ‚Üí "Hello", "C Language"

‚úÖ Example 1: Using const keyword

#include <stdio.h>
int main() {
    const float PI = 3.14;  // constant float
    int r = 5;
    printf("Area = %.2f\n", PI * r * r);
    return 0;
}

‚úÖ Example 2: Using #define

#include <stdio.h>
#define MAX 100   // constant defined at preprocessor level

int main() {
    printf("Maximum value = %d", MAX);
    return 0;
}

Day 3 ‚Äì Operators & Expressions in C

üîπ 1. What are Operators?

Operators are special symbols that perform operations on variables and values. For example: +, -, *, /

An expression is a combination of variables, constants, and operators that produces a result. 

int result = (10 + 5) * 2;   // expression = 30

üîπ 2. Types of Operators

(a) Arithmetic Operators

Used for mathematical calculations.

+ (addition), - (subtraction), * (multiplication), / (division), % (modulus)

#include <stdio.h>
int main() {
    int a = 10, b = 3;
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);  // integer division
    printf("a %% b = %d\n", a % b); // remainder
    return 0;
}

(b) Relational Operators

Compare two values, result is either true (1) or false (0).

==, !=, >, <, >=, <=

int x = 5, y = 10;
printf("%d\n", x > y);   // 0 (false)
printf("%d\n", x < y);   // 1 (true)

(c) Logical Operators

Combine relational conditions.

&& (AND), || (OR), ! (NOT)

int a = 5, b = 10;
printf("%d\n", (a > 0 && b > 0)); // 1 (true)
printf("%d\n", (a > 0 || b < 0)); // 1 (true)
printf("%d\n", !(a > b));         // 1 (true)

(d) Assignment Operators

Used to assign values to variables.

=, +=, -=, *=, /=, %=

int n = 10;
n += 5;  // same as n = n + 5
printf("%d", n); // 15

(e) Increment & Decrement Operators

++ ‚Üí increases value by 1

-- ‚Üí decreases value by 1

int a = 5;
printf("%d\n", a++);  // prints 5, then a = 6
printf("%d\n", ++a);  // a becomes 7, prints 7

(f) Bitwise Operators (Intro Only)

Work at the bit level.

& (AND), | (OR), ^ (XOR), << (Left Shift), >> (Right Shift)

int a = 5, b = 3;  // binary: 5=101, 3=011
printf("%d\n", a & b);  // 1 (001)
printf("%d\n", a | b);  // 7 (111)

(g) Ternary Operator ?:

Short way to write if-else.

int age = 20;
(age >= 18) ? printf("Eligible\n") : printf("Not Eligible\n");

üîπ 3. Operator Precedence & Associativity

Precedence = which operator is evaluated first. Example: * and / have higher precedence than + and -.

Associativity = order when precedence is same.

int result = 10 + 5 * 2;   // result = 20, because * has higher precedence

Day 4 ‚Äì Control Statements in C (Decision Making)

üîπ 1. Introduction

Control statements decide which part of the program should execute based on conditions. In C, the main control statements are:

  • if
  • if-else
  • nested if
  • if-else ladder
  • switch-case
  • goto (not recommended)

üîπ 2. if Statement

#include <stdio.h>
int main() {
    int age = 20;
    if (age >= 18) {
        printf("You are eligible to vote.\n");
    }
    return 0;
}

üîπ 3. if-else Statement

int number = 15;
if (number % 2 == 0) {
    printf("Even number\n");
} else {
    printf("Odd number\n");
}

üîπ 4. Nested if Statement

int a = 10, b = 20, c = 30;
if (a > b) {
    if (a > c)
        printf("a is largest");
} else {
    if (b > c)
        printf("b is largest");
    else
        printf("c is largest");
}

üîπ 5. if-else Ladder

int marks = 75;
if (marks >= 90)
    printf("Grade: A\n");
else if (marks >= 75)
    printf("Grade: B\n");
else if (marks >= 50)
    printf("Grade: C\n");
else
    printf("Fail\n");

üîπ 6. switch-case Statement

#include <stdio.h>
int main() {
    int choice = 2;
    switch(choice) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        default:
            printf("Invalid choice\n");
    }
    return 0;
}

üîπ 7. goto Statement (Intro Only)

Used to jump to a labeled part of the program. ‚ö†Ô∏è Not recommended (makes code hard to read). 

#include <stdio.h>

    int i = 1;
    loop:
        printf("%d ", i);
        i++;
        if(i <= 5) goto loop;
    return 0;
